Frequently Asked Questions (FAQ)

1. How do I install the extension?

To install the extension, follow these steps:

  • Visit the extension page on the Chrome Web Store.
  • Click the "Add to Chrome" button.
  • Confirm the installation by clicking "Add extension".

2. How do I use the extension?

After installing the extension, you can use it by clicking on the extension icon in your browser's toolbar. Follow the on-screen instructions to get started.

3. How do I report a bug or provide feedback?

If you encounter a bug or have feedback, please contact us using the contact information provided below.
